# Frontend Store

Frontend Store es un concepto de tienda online desarrollado como proyecto Front End. El objetivo es mostrar una interfaz limpia, responsiva y bien estructurada utilizando HTML y CSS. Incluye páginas principales de inicio, información sobre la tienda y detalle de producto.

---

## Demo 

https://frontend-tiendatec-lg.netlify.app

---

## Tecnologías utilizadas

- **HTML5** — Maquetación semántica de todas las páginas.
- **CSS3** — Estilos avanzados con CSS (Flexbox / Grid y diseño responsive)
- **Imágenes estáticas** — Contenidos visuales ubicados en la carpeta `img/`.

---

## Estructura del proyecto

frontend-tiendatec-lg/

├── css/

│ └── normalize.css # Base de estilos consistente entre navegadores

│ └── style.css # Estilos globales

├── img/ # Imágenes del sitio

│ └── ...

├── index.html # Página principal

├── nosotros.html # Información de la tienda

└── producto.html # Detalle del producto

---

Buenas prácticas aplicadas

• HTML semántico para mejor estructura y accesibilidad.

• Separación completa entre contenido (HTML) y estilos (CSS).

• Código organizado en carpetas (css/, img/).

• Diseño adaptable a distintas resoluciones.
